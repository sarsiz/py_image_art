# py_image_art
Replicating Images colored/B&amp;W into plots

<p>Install requirements</ps>

```
pip install -r requirements.txt
```

<h5>
Put your Image (to be plotted) in images folder and run the script.
</h5>
<br>
<h5>That's it!</h5>
